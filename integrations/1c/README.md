# Интеграция 1С с Supabase

Файл `SupabaseExchange.bsl` рассчитан на общий серверный модуль 1С.

## Настройка

1. Создайте в конфигурации общий модуль, например `SupabaseОбмен`.
2. Включите свойства модуля:
   - `Сервер`
   - `Вызов сервера`
   - `Экспорт`
3. Перенесите код из `SupabaseExchange.bsl`.
4. В функции `НастройкиSupabase()` замените ключ:

```bsl
Настройки.Вставить("Ключ", "ВСТАВЬТЕ_SUPABASE_SERVICE_ROLE_OR_ANON_KEY");
```

Для полного обмена и удаления данных используйте `service_role key` только на
сервере 1С. Не помещайте service role key в мобильное приложение.

## Таблицы Supabase

Модуль работает с таблицами:

- `fixed_assets`
- `inventory_documents`
- `inventory_document_lines`

Они создаются миграцией:

```text
supabase/migrations/20260509_inventory_documents.sql
```

## Пример вызова из 1С

```bsl
ОсновныеСредства = Новый Массив;
ОсновныеСредства.Добавить(
    SupabaseОбмен.ОсновноеСредствоJSON("fa-001", "Ноутбук Lenovo ThinkPad")
);
SupabaseОбмен.ВыгрузитьОсновныеСредства(ОсновныеСредства);

Документы = Новый Массив;
Документы.Добавить(
    SupabaseОбмен.ДокументИнвентаризацииJSON(
        ТекущаяДата(),
        1001,
        Ложь,
        Ложь,
        "Касымов Ерлан",
        860101300123
    )
);
Результат = SupabaseОбмен.ВыгрузитьДокументыИнвентаризации(Документы);

ДокументSupabase = SupabaseОбмен.НайтиДокументИнвентаризацииПоНомеру(1001);

Строки = Новый Массив;
Строки.Добавить(
    SupabaseОбмен.СтрокаИнвентаризацииJSON(
        ДокументSupabase.id,
        1,
        "fa-001",
        "Ноутбук Lenovo ThinkPad",
        "INV-000124",
        Истина,
        Ложь
    )
);
SupabaseОбмен.ВыгрузитьСтрокиИнвентаризации(Строки);
```

Загрузка документов:

```bsl
ДокументыИзSupabase = SupabaseОбмен.ЗагрузитьДокументыИнвентаризации();
```

Удаление документа:

```bsl
SupabaseОбмен.УдалитьДокументИнвентаризации(1001);
```
